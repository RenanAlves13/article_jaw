# openai

Esta pasta contém notebooks relacionados ao uso de modelos da OpenAI para geração de artefatos de arquitetura.

## Conteúdo

- Notebooks e/ou scripts específicos para execução com a API da OpenAI.

## Pré-requisitos

- Python 3.10+
- Dependências listadas no `requirements.txt` (ou no ambiente do seu notebook).
- Chave de API da OpenAI configurada (ex.: `OPENAI_API_KEY`).

## Como usar

1. Abra o notebook em sua IDE (ex.: VS Code, Jupyter ou Colab).
2. Garanta que a variável de ambiente com sua chave esteja disponível:
   ```bash
   export OPENAI_API_KEY="sua-chave-aqui"
   ```
3. Execute as células na ordem apresentada.

## Observações

- O nome da variável de ambiente pode variar conforme o notebook. Ajuste conforme necessário.
- Se você não for usar os notebooks, não precisa configurar a chave de API para executar `analise.py`.
